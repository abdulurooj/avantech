# 💧 AvanTech – Smart Water Management System

![Logo](./src/assets/logo.png)

## 🚀 About Project
AvanTech is a smart water management system designed to prevent water wastage caused by tank overflow.  
It uses automation and intelligent control systems to manage water levels efficiently.

---

## 🌟 Features

- 🔐 User Authentication (Login / Signup)
- 💧 Smart Water Overflow Prevention
- 📊 Dashboard for Monitoring
- ⚙️ Automation & IoT Integration
- 📱 Responsive UI Design
- 📞 WhatsApp Integration for Contact

---

## 🖥️ Technologies Used

- ⚛️ React.js (Frontend)
- 🎨 Tailwind CSS (Styling)
- 🔗 React Router DOM (Routing)
- 🟢 Node.js & Express (Backend)
- 🗄️ MongoDB (Database)
